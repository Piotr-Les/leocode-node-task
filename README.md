## Description

Node recruitment task.
The API has 3 endpoints and swagger docs available on
`http://localhost:3000/docs`

## Installation

```bash
$ npm install
```

## Running the app

```bash
# watch mode
$ npm run start:dev
```

## How To Test the API

1. run `npm run start:dev`
2. open `http://localhost:3000/docs`
3. use the swagger ui to test endpoints
